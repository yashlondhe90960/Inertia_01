# Inertia Starter Template


## Setup

Download [Node.js](https://nodejs.org/en/download/).

#### 1. Download the zip file of the code.

#### 2. Run the following commands:

```bash
# Install dependencies (only the first time)
npm install

# Run the local server
npm run dev
```
